# Whiteboard
Functionalities:
(1) Chat
(2) Add a note
(3) Delete a note
(4) Create Group
(5) Leave Group
(6) Add a member
(7) Edit Profile
(8) Log-in
(9) Register

# FUTURE WORKS
(1) Freedom Board - real time display of the texts typed by users
(2) Personal Whiteboard - for each user
(3) Real time notifications
